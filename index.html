Robotics

To check communication with your robot, push the button below. You should see "Hello from your robot" if you have a connection. If there is no reponse from the robot, it is likely it is not connected to wifi. Using a laptop or mobile device, check to see if a wireless access point is available with the number of your robot. If so, connect to it and open WIFI manager by going to 192.168.4.1 in a web browser. Then choose a wireless access point available that has internet access. Once connected, try the communication test again.
